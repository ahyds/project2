About the Project

In this project, we choose to grab json format, weather data of random various cities/locations from OpenWeather using API, to create weather data from multiple facets across the globe.


The visualization includes:

  • A Python Flask–powered RESTful API, HTML/CSS, JavaScript, and a database (MongoDB);
  • A dashboard page with multiple charts that update from the same data;
  • Gaugechart with AnyChart JS Library (https://www.anychart.com)

Data Sources

https://openweathermap.org/api

Contents

The data contains the real-time weather information in the following three aspects.


Current UV Index

  • Accessed current UV data for over 500 cities using API.
  • Created a graduated circle map using Leaflet circle vector layers.
  • Filtered the data into three groups based on the UV value and created a layer control in the top-right corner which can toggle the marker on and off.
  • Created a marker for each city with a pop-up that displayed the city name and UV value.

Current Weather

  • Created a dashboard page with multiple charts that update from the same data.
  • Included some level of user-driven interaction, such as dropdowns.
  • Used library “Anychart” to plot 3 gauge charts.
  • Created a dropdown for users to select the city they are interested.
  • Based on the city selected, 3 gauge charts show the temperature, wind and humidity data respectively.
